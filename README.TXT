PROJECT TITLE: Social network for athletes

PURPOSE OF PROJECT: The purpose of this project is to create a basic social
network prototype for athletes.It allows users to create athletes, activities 
(walking, running, biking, swimming, etc.), equipment, and track their performance
statistics like distance traveled and calories burned.
The project demonstrates object-oriented programming concepts such as 
inheritance, encapsulation, abstraction, and basic data processing with Java.

VERSION or DATE: 28 April 2025

HOW TO START THIS PROJECT: 
Make the UML Diagram first

Create a new Athlete
Create new Equipment
Create a new Activity (Regular or Powered)
List all Athletes and Activities
Filter activities by athlete or by mode of transportation
Print activity details
Calculate total distance and calories burned by athlete
